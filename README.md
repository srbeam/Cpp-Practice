# C++ Practice
## Run script (Compile and Execute)
1. go to path of file
2. open terminal
3. Compile: `g++` followed by the `file name` in the terminal and press *enter*. ex:
```
g++ hello.cpp
```
4. Execute: `./` and the machine code `file name` in the terminal and press *enter*. ex:
```
./a.exe
```
